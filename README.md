# 🎮 Tic Tac Tac Game

## Description

A simple and interactive Tic Tac Tac game built using HTML, CSS, and JavaScript.
This project demonstrates basic game logic, user interaction, and DOM manipulation.

## Features

* 🧑‍🤝‍🧑 Two-player gameplay
* ❌⭕ Turn-based system (X and O)
* 🏆 Win and draw detection
* 🔄 Reset game functionality
* 🎯 Simple and clean UI



## Technologies Used

* HTML
* CSS
* JavaScript



## Learning Outcome

This project helped me understand:

* DOM manipulation
* Event handling in JavaScript
* Game logic implementation
* Handling user interactions



## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
